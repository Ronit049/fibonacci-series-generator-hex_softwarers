# Contributing to Fibonacci Series Generator 🤝

Thank you for your interest in contributing to this project! Contributions are welcome and appreciated 🎉

---

## 📌 How to Contribute

### 1️⃣ Fork the Repository

Click on the **Fork** button at the top right of this repository.

---

### 2️⃣ Clone Your Fork

```bash
git clone https://github.com/Ronit049/fibonacci-series-generator-hex_softwarers.git
cd fibonacci-series-generator
```

---

### 3️⃣ Create a New Branch

```bash
git checkout -b feature/new_branch
```

---

### 4️⃣ Make Your Changes

* Improve code quality
* Add new features (e.g., GUI, error handling)
* Fix bugs
* Improve documentation

---

### 5️⃣ Commit Your Changes

```bash
git add .
git commit -m "Add: your meaningful commit message"
```

---

### 6️⃣ Push to GitHub

```bash
git push origin feature/new_branch
```

---

### 7️⃣ Create a Pull Request

* Go to your forked repository
* Click on **Compare & Pull Request**
* Submit your changes for review

---

## ✅ Contribution Guidelines

* Write clean and readable code
* Follow proper naming conventions
* Add comments where necessary
* Keep commits small and meaningful
* Update README if required

---

## 🐛 Reporting Issues

If you find any bugs or issues:

* Open an issue in the repository
* Clearly describe the problem
* Include steps to reproduce it

---

## 💡 Suggestions

Feel free to suggest improvements like:

* Adding GUI (Tkinter)
* Web version (Flask/Django)
* Performance optimization
* More user-friendly features

---

## 📜 Code of Conduct

* Be respectful and professional
* No spam or irrelevant contributions
* Help others learn and grow

---

## 🙌 Thank You

Your contributions help make this project better!

Happy Coding 🚀
