## 🧠 What is Fibonacci Series?

The Fibonacci sequence follows this rule:

Each number = sum of previous two numbers

### Example:

```
0, 1, 1, 2, 3, 5, 8, 13, ...
```

## 📝 Sample Code

```python
n = int(input("Enter number of terms: "))

a, b = 0, 1

for i in range(n):
    print(a, end=" ")
    a, b = b, a + b
```
